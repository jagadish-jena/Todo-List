# Todo List

A modern and responsive Todo List application built with React, Tailwind CSS, and React Icons. It helps users add, update, complete, and organize their daily tasks through a simple interface.

## Features

- Add new tasks
- Edit or delete existing tasks
- Mark tasks as completed
- Show or hide completed tasks
- Save tasks in browser local storage so they remain after a refresh
- Responsive Tailwind CSS design

## Built With

- React
- Tailwind CSS
- React Icons
- UUID
- Vite

## Getting Started

1. Clone the repository.
2. Open the project folder and install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open the local URL shown in the terminal.

## Available Scripts

```bash
npm run dev      # Start the development server
npm run build    # Create a production build
npm run preview  # Preview the production build
npm run lint     # Run ESLint
```
